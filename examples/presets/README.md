# Contoh Preset

File `.toml` siap pakai — buka ScreeniesEditor → **Preset format** →
**Impor .toml** → pilih salah satu file ini.

| File | Untuk apa |
|---|---|
| `jgrp.toml` | Referensi lengkap ber-komentar — sama dengan preset bawaan |
| `contoh-server-kustom.toml` | Template server fiktif dengan aturan & warna berbeda — contek ini untuk server kamu |
| `minimal.toml` | Bukti bahwa field yang tidak ditulis otomatis pakai default |

Panduan lengkap membuat preset: [`docs/PRESETS.md`](../../docs/PRESETS.md).
